# 709. To Lower Case

<br />Given a string `s`, return <em>the string after replacing every uppercase letter with the same lowercase letter</em>.<br />
<br /> <br />
<br />**Example 1:**<br />
```
Input: s = "Hello"
Output: "hello"
```
<br />**Example 2:**<br />
```
Input: s = "here"
Output: "here"
```
<br />**Example 3:**<br />
```
Input: s = "LOVELY"
Output: "lovely"
```
<br /> <br />
<br />**Constraints:**<br />

* `1 <;= s.length <;= 100`

* `s` consists of printable ASCII characters.
