# 1556. Thousand Separator

<br />Given an integer `n`, add a dot (".") as the thousands separator and return it in string format.<br />
<br /> <br />
<br />**Example 1:**<br />
```
Input: n = 987
Output: "987"
```
<br />**Example 2:**<br />
```
Input: n = 1234
Output: "1.234"
```
<br /> <br />
<br />**Constraints:**<br />

* `0 <;= n <;= 2<sup>31</sup> - 1`
