# 504. Base 7

<br />Given an integer `num`, return <em>a string of its **base 7** representation</em>.<br />
<br /> <br />
<br />**Example 1:**<br />
```
Input: num = 100
Output: "202"
```<br />**Example 2:**<br />
```
Input: num = -7
Output: "-10"
```
<br /> <br />
<br />**Constraints:**<br />

* `-10<sup>7</sup> <;= num <;= 10<sup>7</sup>`
